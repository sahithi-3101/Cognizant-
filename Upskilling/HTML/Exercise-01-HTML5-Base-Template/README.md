# Exercise 01 - HTML5 Base Template

## Project Theme

Local Community Event Portal

## Objective

Create a basic HTML5 page using semantic structure and ensure browser compatibility.

## Requirements Implemented

* HTML5 DOCTYPE declaration
* Language attribute (lang="en")
* UTF-8 character encoding
* Semantic elements:

  * nav
  * main
  * footer
* Section comments
* Browser testing in Google Chrome
* Document inspection using Chrome Developer Tools

## Files

* index.html
* Output.png

## Learning Outcome

Learned how to create a valid HTML5 document structure and inspect page elements using browser developer tools.
