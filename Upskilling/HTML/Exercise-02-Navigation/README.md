# Exercise 02 - Navigation and Linking

## Project Theme

Local Community Event Portal

## Objective

Implement intuitive navigation between sections and provide access to an external help document.

## Requirements Implemented

* Created a navigation bar using the `<nav>` element
* Added anchor links for Home, Events, and Contact sections
* Defined matching section IDs
* Created an external Help page using `target="_blank"`

## Files

* index.html
* help.html
* output.png

## Learning Outcome

Learned how to implement internal page navigation using anchor links and how to open external pages in a new browser tab.
